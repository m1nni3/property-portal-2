# Property Oversight & Management Platform

This repository contains the code for an online property oversight and management platform.

## Features
*   Property Portfolio Management
*   Vendor Contact Lists
*   Profit & Loss Tracking
*   Financial Reconciliation
*   Maintenance and Invoice Tracking
*   Automated Report Generation

## Tech Stack
*   **Frontend:** Next.js (React) deployed on Cloudflare Pages
*   **API/Backend:** Cloudflare Workers
*   **Database:** Cloudflare D1
*   **Storage:** Cloudflare R2
*   **CI/CD:** GitHub Actions

## Getting Started
For local development and deployment instructions, please refer to the respective documentation within the `workers/` and `pages/` directories.
